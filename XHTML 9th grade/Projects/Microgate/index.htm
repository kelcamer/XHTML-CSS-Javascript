There are many different kinds of computer devices.

Here is an example of a computer diagram showing the various input and output devices:

The different main parts as shown these diagrams are:
  • memory : Enables a computer to store, at least temporarily, data and programs.
  • mass storage device : Allows a computer to permanently retain large amounts of data. Common mass storage devices include disk drives and tape drives.
  • input device : Usually a keyboard and mouse, the input device is the conduit through which data and instructions enter a computer.
  • output device : A display screen, printer, or other device that lets you see what the computer has accomplished.
  • central processing unit (CPU): The heart of the computer, this is the component that actually executes instructions.
As you can see, there are 4 key parts in the actual cpu register.
  1. The Cache
  2. The RAM
  3. Storage Device Types
  4. Input Sources
