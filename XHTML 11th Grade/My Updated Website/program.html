Chloe goes to a restaurant. Pizza costs $3.00, Spaghetti costs $2.50, and Soda costs $1.00. Let Chloe pick how much of each item she wants, and then tell her the total price including a 7% sales tax.

(Don't forget import Java.util.*)

Steps:

  1. Create five double variables; pizza, spag, and soda, total, and tax. A double variable simply means decimal.

    2. double pizza = 0.0;
    double spag = 0.0;
    double soda = 0.0;
    double total = 0.0;
    double tax = 0.0;

    Keep in mind that the = in Java means assignment, which is substituting the value on the right for the value in the left.
    (Note: It is good programming practice to set all double variables to 0.0, in case you accidentally use them. If you fail to do this and attempt to use them, the compiler will give you a critical error that may possibly crash your computer)

  2. Create a scanner.

    Scanner scanny = new Scanner(System.in);

  3. Tell the user that you want them to first type in the amount of pizzas they want, then the amount of spaghetti, and then the amount of sodas.

    System.out.println("First, type in how many pizzas you want, then how much spaghetti you want, and then how many sodas you want.");

  4. Scan in all three values.

    pizza = scanny.nextDouble();
    spag = scanny.nextDouble();
    soda = scanny.nextDouble();

    (Note: If you are working with strings, the class name is nextLine(), and for integers, it's nextInt())

  5. Calculate the total by multiplying each variable by it's corresponding price, and add them together.

    total = (pizza * 3) + (spag * 2.5) + (soda * 1);

  6. Calculuate Tax by taking the total and multiplying it by .07. (7%)

    tax = total * .07;

  7. Add the tax and the total together and replace the old total with the new total.

    total = tax + total;

  8. Print the result.

    System.out.println("The total is " + total);

    (Note: The + signifys concatentation, which means to connect two pieces of data, NOT addition in this case)
